Learn Cyber Security

Understand ethical hacking basics, network security, Kali Linux, cyber awareness, and security tools — all for educational purposes with a strong focus on ethics and legality.

Cyber security protects computers, networks, and data from unauthorized access and attacks. As digital threats grow, organizations need skilled professionals who understand both offensive and defensive security. CODEZZY teaches these concepts responsibly — for education, certification prep, and career growth only.

⚠ Important: All content on this page is for educational purposes only. Never test security tools on systems you do not own or have explicit written permission to assess. Unauthorized access is illegal and unethical.

Ethical Hacking Basics Roadmap

Phase 1

Security Mindset & Legal Framework

Learn the CIA triad (Confidentiality, Integrity, Availability), types of hackers (white, grey, black hat), and laws like the Computer Fraud and Abuse Act.

Phase 2

Reconnaissance & Scanning

Understand passive and active information gathering, OSINT techniques, port scanning concepts, and vulnerability identification — in lab environments only.

Phase 3

Exploitation & Reporting

Study how vulnerabilities are exploited in theory, practice on platforms like TryHackMe or HackTheBox, and learn professional penetration test reporting.

Kali Linux Roadmap

Week 1

Setup & Navigation

Install Kali in a virtual machine (VirtualBox/VMware), navigate the desktop, update packages, and understand the Linux terminal basics.

Week 2–3

Core Tool Categories

Explore information gathering, vulnerability analysis, web application testing, and password auditing tools — always in isolated lab networks.

Week 4

Lab Practice

Complete guided labs on DVWA (Damn Vulnerable Web Application) and Metasploitable in your own virtual environment.

Network Security Roadmap

Phase 1

Network Fundamentals

Review TCP/IP, subnets, firewalls, and common protocols. Understand how data travels across networks before securing it.

Phase 2

Defense Mechanisms

Firewalls, IDS/IPS, VPNs, network segmentation, and zero-trust architecture principles for modern organizations.

Phase 3

Monitoring & Incident Response

SIEM basics, log analysis, threat detection, and the steps to take when a security incident occurs.

Cyber Awareness & Security Tools

Awareness

Personal Cyber Hygiene

Strong passwords, multi-factor authentication, phishing recognition, safe browsing, and protecting personal data online.

Tools

Essential Security Tools

Learn what Nmap, Wireshark, Burp Suite, and Metasploit do conceptually. Use them only in authorized lab settings for learning.

💡 Career Tip: Certifications like CompTIA Security+, CEH, and OSCP validate your skills. Pair certification study with hands-on lab practice on legal platforms.
